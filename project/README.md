# Netflix Landing Page Clone

A responsive Netflix-inspired landing page built using HTML, CSS, and JavaScript.

## Project Overview

This project recreates the look and feel of the Netflix landing page. It includes a modern user interface, responsive design, movie sections, FAQ accordion functionality, and basic email validation.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Netflix-style hero section
* Navigation bar with logo, language selector, and sign-in button
* Email signup form with validation
* Feature sections highlighting Netflix services
* Movie poster rows with hover effects
* FAQ accordion using JavaScript
* Footer with multiple navigation links
* Smooth animations and interactive UI elements

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your browser.

### Using Live Server

1. Install the Live Server extension in VS Code.
2. Right-click on `index.html`.
3. Select **Open with Live Server**.

## JavaScript Functionality

### Email Validation

### FAQ Accordion

Allows users to expand and collapse frequently asked questions.
